# 駕照筆試模考 UX/UI 優化 (Taiwan Driving Theory Test UX/UI Enhancer)
優化監理服務網駕照筆試模擬考網站，加速練習流程、提供快捷鍵答題方式

[English Documentation](#english-documentation)

---

## 主要功能

* **快速開始**: 
    - 首頁一鍵跳轉答題頁面
* **答題快捷鍵**: 
    - `1` `2` `3` 選答案（是非題 `1` 是 O 、`2` 是 X）
    - `Q` `E`切換上下題、`S` 交卷、`D` 確認、`R` 重新開始
* **防成績頁面自動跳轉**
    - 擋掉了原本 60 秒自動跳轉回首頁，自由訂正錯題
* **影片題自動播放**
    - 機車筆試的影片題腳本會自動開始播放，不用自己按

## 快捷鍵說明
- 注意！因答題頁面切換需加載時間，故短時間內連續點按只有第一下有效

### 1. 首頁 (快速開始)
* `A`：中文汽車
* `S`：中文機車
* `D`：英文汽車
* `F`：英文機車

### 2. 測驗中
* `1` / `2` / `3`：選擇答案
* `Q`：上一題
* `E`：下一題
* `D`：點擊「確定」按鈕
* `S`：交卷
* `H`：回考試首頁

### 3. 成績頁面
* `R`：重新開始測驗 (依照上次的車種設定直接重考)
* `H`：回考試首頁

# Taiwan Driving Theory Test UX/UI Enhancer  <a id="english-documentation"></a>
Optimizes the official MVDIS (Taiwan) driving theory test simulator. Speed up your practice workflow with hotkey support and UI improvements.

---

## Key Features

* **Quick Start**:
    - One-click jump from the homepage to the test page.
* **Hotkeys**:
    - `1` `2` `3` to select answers (For T/F questions: `1` is O, `2` is X).
    - `Q` / `E` to navigate between questions, `S` to submit, `D` to confirm, and `R` to restart.
* **Anti-Redirect (Result Page)**:
    - Blocks the auto-redirect on the results screen, giving you time to review your mistakes.
* **Video Auto-Play**:
    - Automatically plays video questions in motorcycle tests—no manual clicking required.

---

## Hotkeys
- **Note:** Due to loading times between questions, rapid consecutive presses may only register the first input.

### 1. Homepage (Quick Start)
* `A`: Chinese - Automobile
* `S`: Chinese - Motorcycle
* `D`: English - Automobile
* `F`: English - Motorcycle

### 2. During Test
* `1` / `2` / `3`: Select Answer
* `Q`: Previous Question
* `E`: Next Question
* `D`: Click "Confirm" button
* `S`: Submit
* `H`: Back to Homepage

### 3. Result Page
* `R`: Restart Test (Uses your last selected vehicle/language)
* `H`: Back to Homepage
